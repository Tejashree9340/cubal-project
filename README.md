# 🩸 CubaL Emergency Network

CubaL is a smart system that connects blood donors and patients in real-time using Python, MySQL, and data analysis tools.

---

## 🚀 Features

* Donor Registration
* Emergency Request System
* Auto Matching using SQL View
* Queue-based Request Handling (FIFO)
* Data Analysis using Pandas & NumPy
* Graph Visualization (Matplotlib)
* Admin & User Login System

---

## 🧠 Technologies Used

* Python
* MySQL
* Pandas
* NumPy
* Matplotlib
* HTML & CSS

---

## 🔄 How It Works

1. User registers as a donor
2. Patient raises an emergency request
3. System matches donors based on blood group and location
4. Admin can view complete data and analytics

---
